columnize¶

Implement a function “columnize” that formats a list of strings in a compact set of columns of variable width, comparable to what the “ls” command of many unix shells does.

The code is based on cmd.Cmd.columnize from the Python(R) standard library. It is modified and redistributed conforming to the Python Software Foundation License (PSF license).
